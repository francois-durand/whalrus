RuleScore¶

class whalrus.RuleScore(*args, tie_break: whalrus.priority.Priority.Priority = Priority.UNAMBIGUOUS, converter: whalrus.converter_ballot.ConverterBallot.ConverterBallot = None, **kwargs)[source]¶

A voting rule with scores (which are not necessarily numbers).

Each candidate is assigned a score (not necessarily a number), and the the cowinners are the candidates with the best score, in the sense defined by compare_scores().

compare_scores(one: object, another: object) → int[source]¶

Compare two scores.

Parameters

  • one – a score.

  • another – a score.

Returns

0 if they are equal, a positive number if one is greater than another, a negative number otherwise.

property order_¶

Result of the election as a (weak) order over the candidates.

Returns

a list of NiceSet. The first set contains the candidates that have the best score, the second set contains those with the second best score, etc.

property scores_¶

The scores.

Returns

a NiceDict that, to each candidate, assigns a score (non necessarily a number).

property strict_order_¶

Result of the election as a strict order over the candidates.

Returns

a list whose first element is the winner, etc. This may use the tie-breaking rule.

property trailer_¶

“Trailer” of the election.

the “trailer” of the election. This is the last candidate in strict_order_ and also the unfavorable choice of the tie-breaking rule in cotrailers_.

property winner_¶

Winner of the election.

Returns

the winner of the election. This is the first candidate in strict_order_ and also the choice of the tie-breaking rule in cowinners_.
