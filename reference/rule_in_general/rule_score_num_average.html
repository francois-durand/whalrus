RuleScoreNumAverage¶

class whalrus.RuleScoreNumAverage(*args, scorer: whalrus.scorer.Scorer.Scorer = None, default_average: numbers.Number = 0, **kwargs)[source]¶

A voting rule where each candidate’s score is an average of the scores provided by the ballots.

Parameters

  • *args – cf. parent class.

  • scorer – the Scorer. For each ballot, it is in charge of computing its contribution to each candidate’s score.

  • default_average – the default average score of a candidate when it receives no score whatsoever. It may happen, for example, if all voters abstain about this candidate. This avoids a division by zero when computing this candidate’s average score.

  • **kwargs – cf. parent class.

Cf. RuleRangeVoting for some examples.
