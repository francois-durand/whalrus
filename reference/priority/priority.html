Priority¶

class whalrus.Priority(name: str)[source]¶

A priority setting, i.e. a policy to break ties and indifference classes.

Parameters

name – the name of this priority setting.

Typical usage:

>>> priority = Priority.ASCENDING
>>> priority.choice({'c', 'a', 'b'})
'a'
>>> priority.sort({'c', 'a', 'b'})
['a', 'b', 'c']
Variables

  • UNAMBIGUOUS – shortcut for PriorityUnambiguous.

  • ABSTAIN – shortcut for PriorityAbstain.

  • ASCENDING – shortcut for PriorityAscending.

  • DESCENDING – shortcut for PriorityDescending.

  • RANDOM – shortcut for PriorityRandom.

choice(x: Union[set, list], reverse: bool = False) → object[source]¶

Choose an element from a list, set, etc.

Parameters

  • x – the list, set, etc where the element is to be chosen.

  • reverse – if False (default), then we choose the “first” or “best” element in this priority order. For example, if this is the ascending priority, we choose the lowest element. If True, then we choose the “last” or “worst” element. This is used, for example, in RuleVeto.

Returns

the chosen element (or None). When x is empty, return None. When x has one element, return this element.

compare(c, d) → int[source]¶

Compare two candidates.

Parameters

  • c – a candidate.

  • d – another candidate.

Returns

0 if c = d, -1 if the tie is broken in favor of c over d, 1 otherwise.

sort(x: Union[set, list], reverse: bool = False) → Optional[list][source]¶

Sort a list, set, etc.

Parameters

  • x – the list, set, etc.

  • reverse – if True, we use the reverse priority order.

Returns

a sorted list (or None).

The original list x is not modified.

sort_pairs_rp(x: Union[set, list], reverse: bool = False) → Optional[list][source]¶

Sort a list, set, etc. of pairs of candidates (for Ranked Pairs).

Parameters

  • x – the list, set, etc.

  • reverse – if True, we use the reverse priority order.

Returns

a sorted list (or None).

By default, it is in the normal priority order for the first element of the pair, and in the reverse priority order for the second element of the pair.

The original list x is not modified.
