RuleBlack¶

class whalrus.RuleBlack(*args, rule_condorcet: whalrus.rule.Rule.Rule = None, rule_borda: whalrus.rule.Rule.Rule = None, **kwargs)[source]¶

Black’s rule.

Parameters

  • *args – cf. parent class.

  • rule_condorcet – a Rule. Used as the main victory criterion. Default: RuleCondorcet.

  • rule_borda – a Rule. Used as the secondary victory criterion. Default: RuleBorda.

  • **kwargs – cf. parent class.

As a main victory criterion, the Condorcet winner is elected (even if it does not have the highest Borda score):

>>> rule = RuleBlack(ballots=['a > b > c', 'b > c > a'], weights=[3, 2])
>>> rule.rule_condorcet_.matrix_majority_.matrix_weighted_majority_.as_array_
array([[0, Fraction(3, 5), Fraction(3, 5)],
       [Fraction(2, 5), 0, 1],
       [Fraction(2, 5), 0, 0]], dtype=object)
>>> rule.order_
[{'a'}, {'b'}, {'c'}]

When there is no Condorcet winner, candidates are sorted according to their Borda scores:

>>> rule = RuleBlack(ballots=['a > b > c', 'b > c > a', 'c > a > b'], weights=[3, 2, 2])
>>> rule.rule_condorcet_.matrix_majority_.matrix_weighted_majority_.as_array_
array([[0, Fraction(5, 7), Fraction(3, 7)],
       [Fraction(2, 7), 0, Fraction(5, 7)],
       [Fraction(4, 7), Fraction(2, 7), 0]], dtype=object)
>>> rule.order_
[{'a'}, {'b'}, {'c'}]
property cotrailers_¶

“Cotrailers” of the election.

Returns

the set of “cotrailers”, i.e. the candidates that fare worst in the election. This is the last equivalence class in order_. For example, in RuleScoreNum, it is the candidates that are tied for the worst score.

property cowinners_¶

Cowinners of the election.

Returns

the set of cowinners, i.e. the candidates that fare best in the election.. This is the first equivalence class in order_. For example, in RuleScoreNum, it is the candidates that are tied for the best score.

property n_candidates_¶

Number of candidates.

Returns

the number of candidates.

property rule_borda_¶

The Borda rule.

Returns

the Borda rule (once applied to the profile).

property rule_condorcet_¶

The Condorcet rule.

Returns

the Condorcet rule (once applied to the profile).

property rules_¶

The rules (once applied to the profile).

Returns

a list of Rule objects (once applied to the profile).

property strict_order_¶

Result of the election as a strict order over the candidates.

Returns

a list whose first element is the winner, etc. This may use the tie-breaking rule.

property trailer_¶

“Trailer” of the election.

Returns

the “trailer” of the election. This is the last candidate in strict_order_ and also the unfavorable choice of the tie-breaking rule in cotrailers_.

property winner_¶

Winner of the election.

Returns

the winner of the election. This is the first candidate in strict_order_ and also the choice of the tie-breaking rule in cowinners_.
