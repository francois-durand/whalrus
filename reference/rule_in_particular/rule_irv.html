RuleIRV¶

class whalrus.RuleIRV(*args, base_rule: whalrus.rule.Rule.Rule = None, elimination: whalrus.elimination.Elimination.Elimination = None, **kwargs)[source]¶

Instant-Runoff Voting, also known as Alternative Vote, Single Transferable Vote, etc.

Parameters

  • *args – cf. parent class.

  • base_rule – the default is RulePlurality.

  • elimination – the default is EliminationLast with k=1.

  • **kwargs – cf. parent class.

At each round, the candidate with the worst Plurality score is eliminated.

>>> rule = RuleIRV(['a > b > c', 'b > a > c', 'c > a > b'], weights=[2, 3, 4])
>>> rule.eliminations_[0].rule_.gross_scores_
{'a': 2, 'b': 3, 'c': 4}
>>> rule.eliminations_[1].rule_.gross_scores_
{'b': 5, 'c': 4}
>>> rule.eliminations_[2].rule_.gross_scores_
{'b': 9}
>>> rule.winner_
'b'

An example using the tie-break:

>>> rule = RuleIRV(['a > c > b', 'b > a > c', 'c > a > b'], weights=[1, 2, 1],
...                tie_break=Priority.ASCENDING)
>>> rule.eliminations_[0].rule_.gross_scores_
{'a': 1, 'b': 2, 'c': 1}
>>> rule.eliminations_[1].rule_.gross_scores_
{'a': 2, 'b': 2}
>>> rule.eliminations_[2].rule_.gross_scores_
{'a': 4}
>>> rule.winner_
'a'
property cotrailers_¶

“Cotrailers” of the election.

Returns

the set of “cotrailers”, i.e. the candidates that fare worst in the election. This is the last equivalence class in order_. For example, in RuleScoreNum, it is the candidates that are tied for the worst score.

property cowinners_¶

Cowinners of the election.

Returns

the set of cowinners, i.e. the candidates that fare best in the election.. This is the first equivalence class in order_. For example, in RuleScoreNum, it is the candidates that are tied for the best score.

property eliminations_¶

The elimination rounds.

Returns

a list of Elimination objects. The first one corresponds to the first round, etc.

property n_candidates_¶

Number of candidates.

Returns

the number of candidates.

property strict_order_¶

Result of the election as a strict order over the candidates.

Returns

a list whose first element is the winner, etc. This may use the tie-breaking rule.

property trailer_¶

“Trailer” of the election.

Returns

the “trailer” of the election. This is the last candidate in strict_order_ and also the unfavorable choice of the tie-breaking rule in cotrailers_.

property winner_¶

Winner of the election.

Returns

the winner of the election. This is the first candidate in strict_order_ and also the choice of the tie-breaking rule in cowinners_.
