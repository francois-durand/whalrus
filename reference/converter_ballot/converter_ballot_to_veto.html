ConverterBallotToVeto¶

class whalrus.ConverterBallotToVeto(priority: whalrus.priority.Priority.Priority = Priority.UNAMBIGUOUS, order_priority: whalrus.priority.Priority.Priority = None, plurality_priority: whalrus.priority.Priority.Priority = None, veto_priority: whalrus.priority.Priority.Priority = None, one_name_priority: whalrus.priority.Priority.Priority = None)[source]¶

Default converter to a BallotVeto.

Parameters

  • priority – serves as a default value for the other parameters if they are not explicitly mentioned. Default: Priority.UNAMBIGUOUS.

  • order_priority – option passed to BallotOrder.last(). Default: priority.

  • plurality_priority – option passed to BallotPlurality.last(). Default: priority.

  • veto_priority – option passed to BallotVeto.last(). Default: priority.

  • one_name_priority – option passed to BallotOneName.last(). Default: priority.

Typical usages:

>>> converter = ConverterBallotToVeto()
>>> converter(BallotOneName('a', candidates={'a', 'b'}))
BallotVeto('a', candidates={'a', 'b'})
>>> converter(BallotPlurality('a', candidates={'a', 'b'}))
BallotVeto('b', candidates={'a', 'b'})
>>> converter({'a': 10, 'b': 7, 'c':0})
BallotVeto('c', candidates={'a', 'b', 'c'})
>>> converter('a ~ b > c')
BallotVeto('c', candidates={'a', 'b', 'c'})
>>> converter([{'a', 'b'}, 'c'])
BallotVeto('c', candidates={'a', 'b', 'c'})

Use options for the restrictions:

>>> converter = ConverterBallotToVeto(priority=Priority.ASCENDING)
>>> converter('a > b ~ c')
BallotVeto('c', candidates={'a', 'b', 'c'})
